Welcome to the NUbots codebase project page.

Presently, the NUbots codebase runs on Ubuntu 12.04, on the DARwIn-OP humanoid robot, and is used to play robotic football (soccer) in the annual Robocup competition, in the Kid Size Humanoid League

The NUbots codebase is developed by the NUbots team at the University of Newcastle, Australia. The code is released as open source software, under the GNU GPL v2. All documentation relating to the code is also publicly released.
